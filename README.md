# Scrabble Indonesia Online 2-4 Pemain

Cara menjalankan:

```bash
npm install
npm start
```

Buka:
http://localhost:3000

Untuk main di HP dalam Wi-Fi yang sama, buka IP komputer server, misalnya:
http://192.168.1.10:3000

Deploy online: upload folder ini ke Render/Railway/Fly.io/VPS, start command `npm start`.

Kamus: ganti `words.json` dengan daftar kata formal/resmi berizin kalau ingin lebih lengkap.
